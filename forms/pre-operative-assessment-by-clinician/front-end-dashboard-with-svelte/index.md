# Clinician dashboard — SvelteKit + SVAR DataGrid

SvelteKit + Svelte 5 + Tailwind 4 dashboard for reviewing pre-operative
assessments. Uses **SVAR DataGrid** (`@svar-ui/svelte-grid`) with the Willow
theme for sortable columns and dropdown filters.

## Stack

- SvelteKit 2, Svelte 5 runes, TypeScript, Tailwind CSS 4.
- SVAR DataGrid for tabular data.
- Sample-data fallback so the app runs standalone.

## Running

```sh
npm install
npm run dev
```
