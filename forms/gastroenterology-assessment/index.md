# Gastroenterology Assessment

Gastrointestinal symptom evaluation covering upper and lower GI, abdominal pain, liver/pancreas function with red flag detection.

## Scoring system

- **Instrument**: GI Symptom Severity Score
- **Range**: Composite severity score
- **Categories**: Based on symptom frequency, intensity, and red flag presence

## Steps

| #   | Step                      |
| --- | ------------------------- |
| 1   | Demographics              |
| 2   | Chief Complaint           |
| 3   | Upper GI Symptoms         |
| 4   | Lower GI Symptoms         |
| 5   | Abdominal Pain Assessment |
| 6   | Liver & Pancreas          |
| 7   | Previous GI History       |
| 8   | Current Medications       |
| 9   | Allergies & Diet          |
| 10  | Red Flags & Social        |
