# Fall Risk Assessment

Fall risk evaluation using the Morse Fall Scale (MFS) with comprehensive environmental, medication, mobility, cognitive, and sensory assessments for fall prevention planning.

## Scoring system

- **Instrument**: Morse Fall Scale (MFS)
- **Range**: 0-125
- **Categories**:
  - Low Risk: MFS 0-24
  - Moderate Risk: MFS 25-44
  - High Risk: MFS >= 45
- **Severity Levels**:
  - Low: MFS 0-24, minimal risk factors
  - Moderate: MFS 25-44, some risk factors present
  - High: MFS >= 45, multiple risk factors
  - Critical: Recurrent falls with injury, anticoagulated patient, or MFS >= 75

## MFS Components

| Component               | Score Options    |
| ----------------------- | ---------------- |
| History of falling      | 0 / 25           |
| Secondary diagnosis     | 0 / 15           |
| Ambulatory aid          | 0 / 15 / 30      |
| IV / heparin lock       | 0 / 20           |
| Gait                    | 0 / 10 / 20      |
| Mental status           | 0 / 15           |
