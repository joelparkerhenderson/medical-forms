# Psychiatry Assessment

Comprehensive psychiatric evaluation using the GAF (Global Assessment of Functioning) Scale with mental status examination, risk assessment, and capacity evaluation.

## Scoring system

- **Instrument**: GAF Scale (Global Assessment of Functioning)
- **Range**: 1-100
- **Categories**: 91-100 = Superior functioning, 81-90 = Absent or minimal symptoms, 71-80 = Transient symptoms, 61-70 = Mild symptoms, 51-60 = Moderate symptoms, 41-50 = Serious symptoms, 31-40 = Major impairment, 21-30 = Serious impairment, 11-20 = Some danger, 1-10 = Persistent danger
