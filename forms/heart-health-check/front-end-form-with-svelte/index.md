# Heart Health Check: Front End Patient Form With Svelte

Patient-facing 10-step Heart Health Check assessment form. Built with SvelteKit 2.x, Svelte 5 runes, and Tailwind CSS 4.

## Features

- 10-step wizard with progress bar and dynamic routing
- 10 reusable UI components (TextInput, NumberInput, RadioGroup, etc.)
- 10 step-specific components
- QRISK3-based risk scoring engine with heart age calculation
- 20 HHC risk rules and 13 clinical flags
- Reactive assessment store with Svelte 5 runes
- Risk report page with comprehensive results
