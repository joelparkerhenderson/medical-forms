# Heart Health Check: Front End Clinician Dashboard With Svelte

Clinician-facing dashboard for reviewing submitted Heart Health Check assessments. Built with SvelteKit, Svelte 5, Tailwind CSS 4, and SVAR DataGrid.

## Features

- SVAR DataGrid with sortable columns and Willow theme
- Search by NHS number or patient name
- Filter by risk category (Low, Moderate, High)
- Tailwind CSS 4 with NHS colour scheme
- API integration with sample data fallback
