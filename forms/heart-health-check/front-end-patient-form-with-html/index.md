# Heart Health Check: Front End Patient Form With HTML

Patient-facing 10-step Heart Health Check assessment form. Built with plain HTML, CSS, and vanilla JavaScript.

## Features

- 10-step wizard with progress bar
- Landing page with feature cards
- Conditional field visibility based on responses
- Computed values (BMI, TC/HDL ratio) displayed inline
- QRISK3-based risk scoring with heart age calculation
- Risk report with flagged issues and fired rules
- Responsive design with NHS colour scheme
- Print-friendly report layout
