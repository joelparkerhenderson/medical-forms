# Heart Health Check: Front End Clinician Dashboard With HTML

Clinician-facing dashboard for reviewing submitted Heart Health Check assessments. Built with plain HTML, CSS, and vanilla JavaScript.

## Features

- Patient data table with sortable columns
- Search by NHS number or patient name
- Filter by risk category (Low, Moderate, High)
- NHS blue colour scheme with responsive layout
- API integration with sample data fallback
