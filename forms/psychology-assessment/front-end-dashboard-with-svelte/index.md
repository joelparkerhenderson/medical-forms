# Dashboard — SvelteKit + SVAR DataGrid

SvelteKit + Svelte 5 + Tailwind 4 dashboard reviewing DASS-21 submissions.
Uses **SVAR DataGrid** (`@svar-ui/svelte-grid`) with the Willow theme for
sortable columns, severity filters, and risk-flag highlighting.

## Running

```sh
npm install
npm run dev
```
