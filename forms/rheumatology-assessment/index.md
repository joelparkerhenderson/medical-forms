# Rheumatology Assessment

Rheumatological evaluation using the DAS28 (Disease Activity Score in 28 joints) with joint assessment, laboratory results, and extra-articular features.

## Scoring system

- **Instrument**: DAS28 (Disease Activity Score in 28 joints)
- **Range**: Continuous score
- **Categories**: <2.6 = Remission, 2.6-3.2 = Low activity, 3.2-5.1 = Moderate, >5.1 = High activity
