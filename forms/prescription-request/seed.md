# Prescription Request Form

- Patient name, phone, email, patient id (such as UK NHS patient number)
- Clinician name, phone, email, clinician id (such as UK NHS employee number)
- Date
- Medication
- Dosage
- Treatment Instructions
- Options for substitutions, such as different brand names, different generic names, different dosages, etc.
- Is this a new prescription request or a refill prescription request?
- Is this an emergency prescription request or a normal prescription request?
